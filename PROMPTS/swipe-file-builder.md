# Prompt: Swipe File Builder

Build your swipe file systematically by analyzing posts you admire and extracting the patterns.

---

## The Prompt

```
I want to build a swipe file of tweets/posts I admire. 

Analyze this post and extract:
1. THE STRUCTURE: How is it built? (hook, body, payoff, closing)
2. THE IDEA: What is the core idea? (one sentence)
3. THE STRUCTURE TYPE: Is this a list? A story? A paradox? A contrast? A how-to?
4. THE ANGLE: What's the unique perspective or twist?
5. WHY IT WORKS: What psychological pattern does it hit?
6. WHAT I CAN STEAL: What can I adapt for my own voice/niche?

POST TO ANALYZE:
[Paste tweet or post here]
```

---

## How to Use Your Swipe File

Once you have 10-20 posts analyzed:

1. **Structure exchange**: Take the STRUCTURE from one post + the IDEA from another → new tweet
2. **Angle exchange**: Take the ANGLE from one + the IDEA from another → new tweet  
3. **Pattern stacking**: Combine patterns from multiple posts into one

## Example: Structure Exchange

Post A structure: "Nobody talks about [X]. Here's what I learned:"
Post B idea: Walking as a productivity tool

→ New tweet: "Nobody talks about walking. Here's what I learned:"

Post C structure: "3 things about [topic] that nobody mentions"
Post D idea: Building in public

→ New tweet: "3 things about building in public nobody mentions"

## The Rule

You're not copying ideas. You're synthesizing patterns. Your brand = the ideas you associate yourself with.
