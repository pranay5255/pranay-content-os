# Content Reminders

Quick reference for daily use.

---

## Every Morning (5 min)

1. Open `DAILY/YYYY-MM-DD.md`
2. Run idea-generator prompt → fill 3 ideas
3. Run tweet-writer prompt → draft 3 tweets
4. Pick best tweet → edit and post

## Every Evening (5 min)

1. Move today's drafts to `CONTENT/published/`
2. Add sparks/notes to tomorrow's idea bank
3. Check engagement on today's posts

## Every Sunday (The Weekly Reset)

1. Run **content-ideas-generator** (60 ideas) → build pipeline for the week
2. Pick your newsletter theme for next week:
   - Option A: Expand a top-performing tweet from this week
   - Option B: Take a YouTube video topic in your niche (angle + your perspective)
3. Run **deep-posts-generator** on last week's newsletter → sparks for ideas
4. Write next week's newsletter
5. Run **youtube-title-generator** on the newsletter → pick best title
6. Record YouTube video (newsletter as your script)
7. Run **deep-posts-generator** on new newsletter → this week's tweets
8. Clear `DAILY/` folder, archive the week

## Every 2 Weeks

- Batch film 5 reels in one sitting
- Plan next 2 weeks of content themes
- Review STYLE/guide.md → any updates?
- Audit your swipe file → add new posts you admire

---

## The Idea Validation Cycle (The Flywheel)

```
Twitter post (tests the idea)
        ↓
If it performs → YouTube video (deep dive on the topic)
        ↓
Newsletter (full breakdown + research synthesis)
        ↓
More Twitter posts (spinoffs of the winning idea)
        ↓
Repeat
```

---

## Golden Rules (Read When Stuck)

1. **Twitter first** — if it doesn't work as a tweet, it won't work anywhere
2. **Controversy ≠ toxicity** — push to the edge, don't cross it
3. **Specific > generic** — "I lost $12k in 3 months" beats "I lost money"
4. **Your voice > perfect voice** — authentic beats polished every time
5. **Post before you're ready** — done > perfect
6. **Generate first, edit second** — never skip generation
7. **Context is king** — the more you give AI, the better it outputs
8. **2 hours, every day** — that's the entire engine

```
Morning routine (shower, walk, coffee)
        ↓
2 hours at desk
  - Finish 1 section of newsletter
  - Write + schedule 3 social posts
        ↓
Done. Rest of day = freedom.
```

---

## Growth Formula (The 30/70 Split)

Every day, 3 posts:
- **1 post**: Spinoff of your proven winner (exploitation)
- **2 posts**: Experimentation to find the next winner (exploration)

Once a new winner is found → make it a daily spinoff → now you have 2 winners → repeat.

This is the entire growth process on any platform.

---

## When You Have Writer's Block

1. Run **swipe-file-builder** on a post you admire → structure to borrow
2. Run **anatomy-of-viral-posts** → build your personal tweet prompt
3. Run **content-ideas-generator** → 60 ideas, pick one
4. Ask: what did I fail at recently? What did I learn?
5. Ask: what do I disagree with in my industry right now?
6. Ask: what would I tell my past self?

---

## The Prompt Meta-Technique (The Secret Sauce)

When you love something (a tweet, landing page, YouTube video, book):
1. Ask AI to **break it down** (structure + why it works)
2. Treat that as instructions/SOP
3. Feed those instructions into a **"prompt that helps you create prompts"**
4. Get a custom prompt that replicates that thing for YOUR content

This is how you turn anything you love into a repeatable system.
