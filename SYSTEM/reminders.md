# Content Reminders

Quick reference for daily use.

## Every Morning (5 min)

1. Open `DAILY/YYYY-MM-DD.md`
2. Run idea-generator prompt → fill 3 ideas
3. Run tweet-writer prompt → draft 3 tweets
4. Pick best tweet → edit and post

## Every Evening (5 min)

1. Move today's drafts to `CONTENT/published/`
2. Add sparks/notes to tomorrow's idea bank
3. Check engagement on today's posts

## Every Sunday

1. Run weekly review → `SYSTEM/weekly-template.md`
2. Pick 3 best tweets from the week
3. Run newsletter-writer prompt
4. Send newsletter (Monday morning or Sunday night)
5. Clear `DAILY/` folder, archive week

## Every 2 Weeks

- Batch film 5 reels in one sitting
- Plan next 2 weeks of content themes
- Review STYLE/guide.md → any updates?

## Golden Rules (Read When Stuck)

1. **Twitter first** — if it doesn't work as a tweet, it won't work anywhere
2. **Controversy ≠ toxicity** — push to the edge of an opinion, don't cross it
3. **Specific > generic** — "I lost $12k in 3 months" beats "I lost money"
4. **Your voice > perfect voice** — authentic beats polished every time
5. **Post before you're ready** — done > perfect

---

## When You Have Writer's Block

1. Read your last 10 tweets — is there a second angle?
2. Ask: what did I fail at recently? What did I learn?
3. Ask: what do I disagree with in my industry right now?
4. Ask: what would I tell my past self?
5. Steal from things you consumed (books, tweets, convos) and add your take
